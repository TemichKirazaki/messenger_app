# Messenger App

C++ Qt приложение для управления сообщениями с SQLite БД.

## Функции
- Добавление/удаление/поиск сообщений
- Управление пользователями (ФИО, логин, телефон)
- Удаление всех сообщений пользователя

## Технологии
- C++17, Qt6, SQLite3

## Запуск
Windows: `release/run.bat` или установить `packaging/windows/messenger-app.msi`

## Тесты
`tests/run_tests.bat` — 25 тестов, все проходят

Пакеты
- MSI (Windows) 
- DEB (Ubuntu)
- RPM (Fedora)
